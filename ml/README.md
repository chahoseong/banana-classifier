# ML Workspace

이 디렉터리는 학습 스크립트와 모델 관련 자산을 정리하는 작업 공간입니다.

현재 구성:

- `training/`: 학습 스크립트
- `artifacts/`: 모델 파일, feature CSV, 리포트
- `inference/`: 공용 추론 및 feature extraction 코드
- `requirements.txt`: ML 팀 의존성 기준

현재 이동/추가된 자산:

- `training/train_knn.py`
- `inference/base_model.py`
- `inference/feature_extractor.py`
- `inference/ml_baseline.py`
- `artifacts/features.csv`
- `artifacts/baseline_report.png`
- `artifacts/knn_model.pkl`
- `artifacts/scaler.pkl`
