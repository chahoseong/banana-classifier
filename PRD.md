# 🍌 Banana Ripe Classifier - PRD (Product Requirements Document)

## 1. 프로젝트 개요

**프로젝트명:** Banana Ripe Classifier  
**유형:** Computer Vision 기반 이미지 분류 프로젝트  
**팀 구성:** 3명  

웹캠(노트북 카메라)에 바나나 사진을 보여주면, AI 모델이 바나나의 숙성도를 자동으로 판별하여 결과를 화면에 표시하는 애플리케이션입니다.

---

## 2. 목표

- 웹캠 실시간 입력으로 바나나 이미지를 캡처한다.
- AI 모델이 바나나를 아래 4가지 카테고리 중 하나로 분류한다.
- 분류 결과와 신뢰도(confidence)를 사용자에게 즉시 표시한다.

### 분류 카테고리

| 카테고리 | 설명 |
|---|---|
| **unripe** | 미숙성 (초록색이 남아 있는 상태) |
| **ripe** | 숙성 (노란색, 먹기 적절한 상태) |
| **overripe** | 과숙성 (갈색 반점이 나타난 상태) |
| **dispose** | 폐기 (심하게 변색, 먹을 수 없는 상태) |

---

## 3. 데이터

- **총 이미지 수:** 약 600장
- **이미지 특성:** Aspect Ratio 및 해상도가 제각각
- **폴더 구조:** 숙성도별 폴더에 이미지가 분류되어 있음
- **전처리 필요 사항:**
  - 종횡비가 다른 이미지를 패딩(Padding) 또는 크롭(Crop)으로 규격화
  - 데이터 증강(Data Augmentation)으로 학습 데이터 부족 보완

---

## 4. 시스템 아키텍처

```
[웹캠] → [프론트엔드 (Gradio/Streamlit)] → [추론 엔진 (AI 모델)] → [분류 결과 출력]
```

### 핵심 설계 원칙

1. **프론트엔드 교체 가능:** 현재는 Gradio/Streamlit으로 빠르게 구현하되, 추후 모바일 앱 등으로 교체할 수 있도록 프론트엔드와 추론 엔진을 분리한다.
2. **다중 모델 지원:** 여러 모델(ML 베이스라인, 딥러닝 모델 등)을 플러그인 방식으로 교체·비교할 수 있도록 구현한다.

---

## 5. 참여자 역할

### 👤 팀원 1: 데이터 엔지니어 (데이터 파이프라인 & 베이스라인 모델)

- 이미지 전처리 파이프라인 구축 (리사이즈, 패딩, 크롭, 정규화)
- 색상 히스토그램, 질감 등 특징(Feature) 추출
- ML 베이스라인 모델 개발 (LGBM, KNN, 로지스틱 회귀 등)
- 기준 정확도(Baseline Accuracy) 확보

### 👤 팀원 2: 모델 엔지니어 (딥러닝 추론 엔진)

- 데이터 증강(Data Augmentation) 적용
- 전이 학습(Transfer Learning) 기반 딥러닝 모델 개발 (ResNet, MobileNet 등)
- 하이퍼파라미터 튜닝 및 과적합 방지
- 최종 모델 성능 최적화

### 👤 팀원 3: 프론트 엔지니어 (UI 연동 & 시스템 통합)

- Gradio 또는 Streamlit 기반 웹캠 UI 구축
- 모델 통합 인터페이스 구현 (모델 교체 가능한 구조)
- 실시간 추론 속도(FPS) 및 정확도 비교 평가
- 최종 배포 환경 구성

## 6. 기술 스택

| 구분 | 기술 |
|---|---|
| 언어 | Python |
| 프론트엔드 | Gradio 또는 Streamlit |
| ML 모델 | scikit-learn (KNN, Logistic Regression), LightGBM |
| DL 모델 | PyTorch 또는 TensorFlow (ResNet, MobileNet) |
| 이미지 처리 | OpenCV, Pillow |
| 데이터 증강 | torchvision.transforms 또는 albumentations |

---

## 7. 제약 조건

1. **프론트엔드 분리:** 프론트엔드는 추후 모바일 앱 등으로 교체할 수 있도록 추론 엔진과 독립적으로 구현한다.
2. **다중 모델 지원:** 여러 모델을 쉽게 교체하여 사용할 수 있는 인터페이스를 제공한다.
3. **노트북 환경 호환:** 일반 노트북(GPU 없는 환경)에서도 실시간 추론이 가능해야 한다.
